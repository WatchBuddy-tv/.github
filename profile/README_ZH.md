**WatchBuddy** 是一个模块化生态系统，让你无论身在何处，都能与朋友同时观看视频。
我们的目标是提供一个统一的同步桥梁，覆盖网页端、移动端和桌面端。

## 开发生态：ExampleProvider

WatchBuddy 建立在**去中心化且可扩展**的架构之上。你不受预定义内容限制；你可以创建自己的内容提供方，并轻松接入 WatchBuddy 生态。

[ExampleProvider](https://github.com/WatchBuddy-tv/ExampleProvider) 项目就是官方蓝图：

- 即插即用：只需把提供方地址添加到 WatchBuddy，即可开始观看。
- 独立开发：在本地环境中开发和测试抓取器，无需依赖核心系统。
- 基于 KekikStream：使用现代链接提取工具解析媒体 URL。

## 核心特性

- 完整同步：所有参与者实时同步播放。
- 多样媒体支持：原生支持 M3U/HLS 流与直链视频 URL。
- 不托管媒体：WatchBuddy 是同步桥梁，不托管、不存储、不分发媒体内容。
- 云同步：可选资料同步，跨设备保留收藏和历史记录。
- 实时互动：观看时支持实时聊天与反应分享。
